# Agents Design

## 1️⃣ Ingestion Agent

### Responsibilities
- Store uploaded files
- Extract text
- Generate embeddings
- Chunk documents

---

## 2️⃣ Extraction Agent

### Responsibilities
- Extract:
  - Expiry dates
  - Financial terms
  - Key parties
  - Clause categories

---

## 3️⃣ Risk Agent

### Responsibilities
- Identify unusual or risky clauses
- Compare with known legal patterns
- LLM-based classification

---

## 4️⃣ Obligation Agent

### Responsibilities
- Create reminders
- Generate calendar events
- Notify users

---

## 5️⃣ QA Agent

### Responsibilities
- Answer document-related questions
- Perform semantic search
- Cross-document reasoning
